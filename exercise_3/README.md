# Exercise 3 – Forest Fire Cellular Automaton Driven by NASA FIRMS Data

## Goal
Create a forest fire cellular automaton driven by NASA FIRMS hotspot detections.

## Minimum implementations
- Serial automaton
- MPI domain-decomposed automaton

## Suggested experiments
- Several grid sizes or simulation horizons
- Runtime comparison serial vs MPI
- Snapshots or animation of fire spread
- Discussion of scientific limitations

## Folder notes
- `serial/`: serial simulation
- `parallel_mpi/`: MPI version
- `data/`: FIRMS data and preprocessing scripts
- `outputs/`: figures, animations, timing logs
