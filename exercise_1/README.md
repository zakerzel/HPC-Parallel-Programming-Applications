# Exercise 1 - Parallel Matrix Multiplication

This exercise compares five strategies for matrix multiplication:

1. Serial dense baseline
2. Multiprocessing by rows
3. Multiprocessing by columns
4. Multiprocessing by blocks
5. MPI scatter/gather version
6. Recursive Strassen hybrid

## Files

- `src/matmul_core.py`: shared implementations
- `src/benchmark_dense.py`: dense synthetic benchmarks
- `src/benchmark_sparse.py`: sparse matrix experiments
- `src/matmul_mpi.py`: MPI distributed-memory implementation
- `results/`: CSVs and figures generated by benchmarks

## Run

```bash
python src/benchmark_dense.py
python src/benchmark_sparse.py
mpiexec -n 4 python src/matmul_mpi.py
```
